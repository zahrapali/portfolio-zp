Terry Fox Foundation

Role:

Designer and Developer.

OBJECTIVE:

Design and Develop an engaging website for easy access to resources, events and donation options.

PLATFORM:

React + Wordpress.

User Types Included:

  1. Run Organizers
  2. Run Particpants
  3. Corporations
  4. Donors

User Research:

Learning about the key objectives for each user group was imperative to create the user journey for each. While doing so, it enabled us to make specific design decisions to favour those users' needs. We were able to justify the "less is more" notion when it comes to websites.

Challenges:

Stakeholders. Justifying our design choice to various stakeholders with different objectives was a challenge. Leveraging the UX research and user journey flows to defend our design decisions through storytelling and prototyping to satisfy the stakeholders while also building a product with the users in mind.
