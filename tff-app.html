TFF Run App

Role:

Product Manager + designer.

OBJECTIVE:

Terry Fox Run App! The Terry fox foundation is known for their yearly community and school runs! Due to the pandemic, the TFF team approach scratch to come up with a fun and user-friendly solution to make sure individuals could participate in the runs while still maintaining social distancing measures.

PLATFORM:

Movespring + PHP Back-End.

User Types Included:

  1. TFF Organizers
  2. Community Run Organizers
  3. Pariticpants
  4. Donors

User Research:

Cue an app dedicated to creating challenges and community-based groups to work towards a fundraising goal. The Terry Fox Run app was custom created. We hosted the Mount Terry Fox Virtual Climb and the community-run. Prior to the run, the team at TFF created a community-based competition to enable participants to still complete their activities amongst their friends but in a virtual way. We custom-designed the map to include Canadian landmarks and create a run and interact way to “run across Canada”. And worked seamlessly with the Movespring Team to ensure proper design for the app and user functionality.
