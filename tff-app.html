TFF Run App

Role:

Product Manager + Designer.

OBJECTIVE:

Terry Fox Run App! The Terry Fox Foundation is known for their yearly community and school runs! Due to the pandemic, the TFF team approach Scratch to come up with a fun and user-friendly solution to make sure individuals could participate in the runs while still maintaining social distancing measures.

PLATFORM:

Movespring + PHP Back-End.

User Types Included:

  1. TFF Organizers
  2. Community Run Organizers
  3. Pariticpants
  4. Donors
  5. Schools

Product Breakdown:

Cue an app dedicated to creating run challenges and community-based groups to work towards a fundraising goal. The Terry Fox Run app was custom created with the Team at Movespring. We hosted the Mount Terry Fox Virtual Climb and the Community-Run on the app. Leading up toe the Community run, the team at TFF created a community-based competition to achieve badges and work with collegues, friends, families to clock in KM for pledges. We custom-designed the map to include Canadian landmarks and create a route that “ran across Canada” in the same way Terry Fox did.

The Terry Fox Foundation raised a record breaking donations through the App.
